# Feature Backlog

Product feature checklist for a general-purpose dictation/document workspace.
This list excludes infrastructure hardening tasks tracked in `docs/TODO_BACKLOG.md`.

## Immediate Priorities

- [ ] Document organization
  - [ ] Folders or projects
  - [ ] Tags
  - [ ] Favorites or pinned documents
  - [ ] Recent documents
  - [ ] Archive or trash
  - [ ] Search across all documents
- [ ] Import and export
  - [ ] Import `.docx`
  - [ ] Import `.txt`
  - [ ] Import `.md`
  - [ ] Export `.docx`
  - [ ] Export `.txt`
  - [ ] Export `.md`
  - [ ] Batch export
  - [ ] Print-friendly mode
- [ ] Document history and recovery
  - [ ] Version history
  - [ ] Restore previous version
  - [ ] Autosave recovery snapshots
  - [ ] Compare versions
  - [ ] Trash restore flow
